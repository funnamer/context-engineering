# 🧪 self-harness 内测招募 | 一起来构建更 robust 的 AI Agent

> 从 Prompt Engineering 到 Context Engineering,再到 Harness Engineering——我们想和你一起探索大模型时代 Agent 系统的工程化范式。

self-harness 是一本面向开发者的开源实战教程,目前进入 Beta 内测阶段。教程不仅系统讲解了三种工程的演进逻辑,还附带了一个可运行的 miniMaster 项目——一个最小化的 harness 实现,帮助你快速体验 harness 系统,包含动态工作记忆、三层嵌套循环(Plan → Generator → Validate)和完整的工具注册体系。

## 🎁 内测你能获得什么

- **深入理解 Agent 工程**:从宏观角度看待 Prompt、Context、Harness 这些智能体工程,理解它们为什么存在、解决了什么问题,对 Agent 工程有更深的认识
- **实战 Multi-Agent 系统**:运行一个最小化但完整的 Multi-Agent 系统,快速体验和学习如何通过 Harness 工程保证系统稳定性
- **影响项目发展**:直接对教程和代码提建议,影响最终版本
